## Como hospedar (GitHub Pages)

```bash
# 1. Crie o repositório no GitHub com o nome:
#    Roseleyne.github.io   (para URL raiz)
#    OU qualquer nome como "portfolio" (para URL /portfolio)

# 2. Faça o clone e copie os arquivos
git clone https://github.com/Roseleyne/portfolio.git
cd portfolio

# 3. Adicione os arquivos
git add .
git commit -m "feat: portfolio inicial"
git push origin main

# 4. Ative o GitHub Pages:
#    Repositório → Settings → Pages → Source: Deploy from branch → main → / (root)
#    Aguarde ~1 minuto → URL disponível!
```
